## Технические детали реализации

### Фронтенд
- ✅ **Next.js (TSX)**: Для создания серверного рендеринга и оптимизации SEO
- ✅ **Tailwind CSS**: Для быстрой разработки стилей
- ⚠️ **shadcn/ui**: Для использования готовых компонентов интерфейса (не найдено явных импортов)
- ❌ **React Query**: Для управления состоянием и кэширования данных
- ❌ **Framer Motion**: Для анимаций и улучшения UX
- ❌ **Recharts/D3.js**: Для визуализации данных и прогресса

### Бэкенд
- ✅ **Supabase**: Для хранения данных, аутентификации и авторизации
- ❌ **Edge Functions**: Для серверной логики и интеграции с ИИ-сервисами
- ❌ **Webhooks**: Для интеграции с внешними сервисами

### ИИ-компоненты
- ❌ **OpenAI API**: Для генерации подсказок и объяснений
- ❌ **TensorFlow.js**: Для клиентской обработки данных и персонализации
- ❌ **Langchain**: Для работы с большими языковыми моделями

### Инфраструктура
- ❌ **Vercel**: Для хостинга и автоматического деплоя
- ❌ **GitHub Actions**: Для CI/CD
- ❌ **Sentry**: Для мониторинга ошибок
- ❌ **Plausible/Umami**: Для аналитики без нарушения конфиденциальности
